AppFramework UI

Mobile App UI framework

AppFramework UI is the User Interface/User Experience library for mobile applications. It uses HTML5 and CSS3 for animations and transitions. It provides javascript APIs to control the UI elements. It also provides vector icons and native looking UI themes for iOS, Android, Windows Phone and Blackberry.

Remote Data Demo

Get Remote Data
Get Remote Data Ext


Methods

You can use this method for making background POST/GET requests for remote data.

intel.xdk.device.getRemoteData(url, requestMethod, requestBody, successCallback, errorCallback)
intel.xdk.device.getRemoteDataWithID(url, requestMethod, requestBody, successCallback, errorCallback, uniqueIdentifier)
var param = new intel.xdk.Device.RemoteDataParameters();

//param : {url, header, body, subject, url, method}

intel.xdk.device.getRemoteDataExt(param);

Event

Use this event listener to get the remote data from getRemoteDataExt().

document.addEventListener( "intel.xdk.device.remote.data", function(e){alert(e.response)}, false)
